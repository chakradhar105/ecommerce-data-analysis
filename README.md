# 🛒 E-commerce Data Analysis Project

## 📌 Overview
This project performs end-to-end data analysis on an e-commerce dataset to extract meaningful business insights.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🧹 Data Cleaning
- Removed missing values (CustomerID, Description)
- Removed duplicate records
- Filtered invalid data (negative Quantity and UnitPrice)
- Converted InvoiceDate to datetime format
- Created a new column: **TotalPrice**

## 📊 Key Insights
- United Kingdom generates the highest revenue
- Identified top-selling products by quantity and revenue
- Observed monthly sales trends

## 📈 Visualizations
- Top 5 Countries by Sales
- Monthly Sales Trend
- Top Products Analysis

## 💡 Business Insights
- The United Kingdom contributes the majority of total revenue.
- A small number of products generate a large portion of sales.
- Sales show variation across months indicating seasonal trends.
  
## 🚀 Project Outcome
This project demonstrates a complete data analysis workflow from raw data to actionable insights.
